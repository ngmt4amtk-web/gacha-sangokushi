## 1. 司馬懿 深掘り

### 1-1. 出自と家系

司馬懿（字は仲達）、179年生まれ。**河内郡温県孝敬里**の出身。司馬氏は楚漢戦争時代の殷王・司馬卬から十二代を経た名族であり、代々高官を輩出した。

父の**司馬防**（字は建公）は京兆尹を務めた人物。司馬防の八人の息子はすべて字に「達」の字を持ち、**「司馬八達」**と称された。

| 順 | 名 | 字 |
|----|------|------|
| 長男 | 司馬朗 | 伯達 |
| **次男** | **司馬懿** | **仲達** |
| 三男 | 司馬孚 | 叔達 |
| 四男 | 司馬馗 | 季達 |
| 五男 | 司馬恂 | 顕達 |
| 六男 | 司馬進 | 恵達 |
| 七男 | 司馬通 | 雅達 |
| 八男 | 司馬敏 | 幼達 |

『晋書』宣帝紀によれば、司馬懿は幼少より聡明で、曹植は司馬懿を評して**「魁傑雄特、秉心平直」**（傑出して偉大であり、公正な心を持つ）と述べた。

### 1-2. 「狼顧の相」の逸話

『晋書』宣帝紀の記述（原文）:

> **帝内忌而外寛、猜忌多権変。魏武察帝有雄豪志、聞有狼顧相、欲験之。乃召使前行、令反顧、面正向後而身不動。又嘗夢三馬同食一槽、甚悪焉。因謂太子丕曰：「司馬懿非人臣也、必預汝家事。」太子素与帝善、毎相全佑、故免。帝於是勤於吏職、夜以忘寝、至於芻牧之間、悉皆臨履、由是魏武意遂安。**

**現代語訳**: 司馬懿は内心では猜疑深く、外面は寛大に見せかけ、策謀に長けていた。曹操は司馬懿に雄大な志があることを察し、「狼顧の相」があると聞いて試そうとした。そこで司馬懿を呼んで前を歩かせ、後ろを振り返らせたところ、**顔は真っ直ぐ後ろを向いているのに体は動かなかった**。また曹操はかつて「三頭の馬が同じ飼い葉桶（槽＝曹と同音）から食べる」夢を見て、ひどく嫌った。そこで太子の曹丕に「司馬懿は人臣の器ではない。必ずお前の家のことに干渉するだろう」と警告した。しかし曹丕は司馬懿と親しかったため、常にかばい守ったので、司馬懿は難を免れた。司馬懿はこれ以後、吏務に精励し、夜も寝ずに働き、家畜の世話に至るまで自ら立ち会ったので、曹操はようやく安心した。

**ゲームへの活用**: この逸話は司馬懿の「隠忍自重」「狡猾」というキャラクター性を象徴するもの。UR/SSR司馬懿の説明文やスキル名に使える（例: スキル名「狼顧」、「三馬同槽」）。

また、晋の明帝・司馬紹がこの逸話を聞いた際のエピソードも重要:

> **「若如公言、晋祚復安得長遠！」**（もしその言葉の通りならば、晋の国運がどうして長く続きましょうか！）

明帝自身が、祖先の行いに恥じ入ったという挿話である。

### 1-3. 曹操への仕官拒否

**建安6年（201年）**、司馬懿は河内郡の上計掾に推挙され、当時司空であった曹操がその才能を聞いて出仕を求めた。

『晋書』宣帝紀によれば、司馬懿は**漢王朝の衰微を見て、曹操に屈することを良しとせず、「風痺」（リウマチ＝足の病）を理由に辞退した**。

曹操は疑い、夜中に人を遣わして刺客を送ったが、司馬懿は臥したまま微動だにしなかった。この話は司馬懿の忍耐力と演技力を示す逸話として知られる。

**建安13年（208年）**、丞相となった曹操は改めて司馬懿を文学掾に任命し、**「もしまた躊躇するようであれば、捕らえてでも連れてこい」**と命じた。このため司馬懿はやむを得ず出仕した。

**別説**:『北堂書鈔』に引く『魏略』では、曹洪が司馬懿に補佐を求めたが、司馬懿は曹洪との交際を恥とし、仮病を使って杖をついたとする。

**史料批評**: 『晋書』は唐代の編纂であり、晋王朝の祖先を美化・神秘化するために誇張された逸話が多く含まれる。仕官拒否の逸話も、「漢への忠節」を演出するための後付けの可能性が高いと指摘されている。

### 1-4. 曹丕との関係と信頼

司馬懿と曹丕の関係は三国時代における最も重要な君臣関係の一つである。

**曹操の後継者争い**: 曹丕と曹植が後継を争った際、司馬懿は一貫して曹丕を支持した。曹操が司馬懿の野心に気づいて曹丕に警告しても、**「太子素与帝善、毎相全佑」**（曹丕は日頃から司馬懿と親しく、常にかばい守った）とあるように、曹丕は司馬懿を守り続けた。

**220年**、曹操が死去。曹丕は漢の献帝から禅譲を受けて魏の初代皇帝（文帝）となり、引き続き司馬懿を重用した。

**226年**、曹丕が崩御する際、**遺詔**により司馬懿を**顧命大臣**に任命した。遺詔の相手は以下の四名:

| 名 | 役割 |
|----|------|
| 曹真 | 大将軍 |
| 曹休 | 大司馬 |
| 陳羣 | 司空 |
| **司馬懿** | 撫軍大将軍 |

曹丕は太子の曹叡に対し「この四人は信頼に足る」と述べて後事を託した。この遺詔が司馬懿の政治的地位を決定的にした。

曹叡もまた司馬懿を信頼し、**「西方有事、非君莫可付者」**（西方に事あらば、君でなければ任せられぬ）と評した。

### 1-5. 対蜀戦

#### 街亭の戦い（228年）

**重要な史実の訂正**: 正史『三国志』では、**街亭で馬謖を破ったのは張郃であり、司馬懿ではない**。『三国志演義』では読者の興味を引くために張郃を司馬懿に差し替えているが、これは小説の創作である。

正史の記述: 第一次北伐において諸葛亮は馬謖を前線の要衝・街亭に派遣した。馬謖は諸葛亮の命に背き水辺を捨てて山上に陣を構えた。副将の王平が何度も諫めたが馬謖は聞き入れなかった。**魏の張郃は蜀軍の水を汲む道を断ち、山上に孤立した蜀軍を攻撃して大いに打ち破った**。

その後、諸葛亮は馬謖を軍法に従い処刑し（「泣いて馬謖を斬る」）、自らも三階級降格した。

#### 五丈原の対陣（234年）

諸葛亮の第五次北伐。五丈原（現在の陝西省宝鶏市岐山県）にて、蜀軍と魏軍が対陣した。

**100日余りの膠着**: 司馬懿は諸葛亮の知略を深く恐れ、守りに徹して決して戦おうとしなかった。

**女物の髪飾りの挑発**: 『晋書』宣帝紀には以下の記述がある:

> **亮数挑戦、帝不出。因遺帝巾幗婦人之飾。**

（諸葛亮は何度も挑戦したが、司馬懿は出陣しなかった。そこで司馬懿に**巾幗婦人之飾**（女性の髪飾り）を贈った。）

これは「お前は女のように臆病だ」という侮辱であったが、司馬懿はなおも耐え忍んだ。

**諸葛亮の死**: 建興12年（234年）8月、諸葛亮は陣中にて病没。享年54歳。

#### 「死せる孔明、生ける仲達を走らす」

蜀軍の撤退を知った魏軍は追撃に出たが、蜀軍が反撃の陣形を見せると**司馬懿は慌てて軍を退いた**。

> **百姓為之諺曰：「死諸葛走生仲達」。帝聞而笑曰：「吾便料生、不便料死也。」**

（民はこれを揶揄して「死せる諸葛、生ける仲達を走らす」と言った。司馬懿はこれを聞いて笑い、**「私は生者のすることは推し量れるが、死者のすることは推し量れない」**と答えた。）

**正史と演義の違い**: 『三国志演義』では諸葛亮が生前に木像を用いて司馬懿を欺く策略があるが、これは正史にはない創作。

### 1-6. 遼東公孫淵討伐（238年）

**背景**: 公孫氏は三代にわたって遼東を事実上独立支配していた。公孫淵は呉の孫権と密かに通じ、遼東王を自称して魏に反旗を翻した。

**出陣**: 曹叡は司馬懿に**4万余の兵**を与え、公孫淵征伐を命じた。

**戦術的機動**:
- 公孫淵は遼隧に数十里の塹壕を掘り、防衛態勢を敷いた
- 司馬懿は正面攻撃を避け、**陽動で東南に退却したと見せかけ**、急転して公孫淵の国都・襄平に直行
- 敵将の卑衍・楊祚が救援に戻ると、司馬懿がこれを待ち受けて大敗させた

**降伏と処刑**: 長雨で兵糧が底を突いた公孫淵は和議を求めて使者（王建・柳甫）を送ったが、**司馬懿は二人をその場で斬首**。公孫淵と子の公孫脩は捕らえられて斬首された。

**戦後の虐殺**: **遼東の15歳以上の成年男子7000人余が虐殺され、その首を積み上げて京観（高楼）が築かれた**。これは司馬懿の残忍な一面を象徴する出来事である。

### 1-7. 高平陵の変（249年）

三国時代の歴史を決定づけた最大のクーデター。

**背景**:
- 曹叡の死後（239年）、幼帝・曹芳が即位
- 曹爽と司馬懿が共に顧命大臣として輔政
- 曹爽が権力を独占し、司馬懿を名誉職（太傅）に棚上げ
- 司馬懿は**病と老衰を装い、朝廷から退いたふりをした**

**詐病の演技**: 司馬懿は曹爽の使者・李勝が訪問した際、わざと衣を落とし、粥をこぼし、耳が聞こえない振りをするなど、**徹底的にボケ老人を演じた**。李勝はこれを見て「司馬公はもうすぐ死ぬ」と曹爽に報告し、曹爽は完全に油断した。

**クーデター決行**: 正始10年（249年）1月6日。

| 時系列 | 出来事 |
|--------|--------|
| 1月6日 | 曹芳が高平陵（明帝の陵墓）参拝に出発。曹爽兄弟も随行 |
| 同日 | 司馬懿が即座に行動。宮中に参内し**郭太后**に上奏 |
| 同日 | 司馬師・司馬孚が洛陽の宮城を制圧 |
| 同日 | 高柔・王観に命じ曹爽・曹羲の軍営を制圧 |
| 同日 | 司馬懿が蒋済とともに洛水の岸辺に布陣 |
| 数日間 | **曹爽を「免官するだけだ」と説得して降伏させる** |
| 1月10日 | 曹爽らに謀反の罪を着せ、**一族郎党皆殺し** |

**粛清された主な人物**:
- 曹爽とその兄弟（曹羲・曹訓・曹彦）
- 何晏、桓範、丁謐、畢軌、李勝、鄧颺
- いずれも**三族皆殺し**

**歴史的意義**: この政変により、魏の実権は事実上司馬氏に移った。以後、司馬師・司馬昭・司馬炎と三代にわたって司馬氏が権力を掌握し、最終的に晋王朝の建国に至る。高平陵の変は「三国時代の終わりの始まり」と言える。

### 1-8. 晋書・裴松之注での評価

#### 『晋書』宣帝紀における評価

『晋書』は唐代に**唐太宗（李世民）の勅命**により編纂された。特筆すべきは、**唐太宗が自ら宣帝紀の論賛を執筆した**ことである。

唐太宗は司馬懿を以下のように批判的に評した:
- 遼東での過剰な虐殺について、**「一国の開祖としてこれほどの殺戮をなすは、その報いが子孫に及ぶ」**と論じた
- 高平陵の変での曹爽への背信については、道義的に問題があるとした

しかし同時に、司馬懿の軍事的才能は高く評価された。

#### 正史『三国志』における扱い

陳寿の『三国志』には司馬懿の独立した伝がない。これは陳寿が晋の臣下であり、晋の祖である司馬懿を臣下として「伝」に書くことは不敬にあたるためである。そのため、**司馬懿に関する詳細な記録は『晋書』宣帝紀が最も重要な一次資料**となる。

ただし、『晋書』は「信憑性の低い逸話が多い」と批判されることも多く、清朝の考証学では特に「狼顧の相」などの記述が事実とは認め難いと論難された。

#### 裴松之注での批評

裴松之は『三国志』に膨大な注を付した南朝宋の学者で、司馬懿に関しても複数の異説・補足を収録している。裴松之は史料の信頼性に厳しい態度で臨み、誇大な逸話に対しては疑義を呈することが多い。

**ゲームへの活用ポイント**: 司馬懿のキャラクター描写では、以下の二面性を強調する:
1. **忍耐・慎重・知略**: 狼顧の相を隠し通し、詐病で曹爽を欺き、五丈原で諸葛亮の挑発に耐えた
2. **冷酷・残忍**: 遼東の虐殺、高平陵の変での背信と皆殺し

## 2. 全8章 ストーリーテキスト用 史実詳細

### 第1章: 黄巾の乱（184年）

#### 張角の太平道

張角は冀州鉅鹿郡の人。『太平清領書』に基づく道教的な悔過（罪を悔いること）による治病を行い、**太平道**を広めた。信徒は数十万に達し、**36個の「方」**（軍事組織的な単位）に編成された。

#### 「蒼天已死、黄天当立」

> **蒼天已死、黄天当立、歳在甲子、天下大吉**

（蒼き天はすでに死に、黄なる天がまさに立つべし。歳は甲子にあり、天下は大いに吉なり。）

「蒼天」は漢王朝（五行思想で「火徳」→「土徳」への交替を主張）を指し、「黄天」は黄巾軍の新秩序を指す。各地の役所の門に「甲子」の二文字が書かれ、蜂起を呼びかけた。

#### 蜂起と挫折

184年（光和7年、干支は甲子）に蜂起予定。先に荊州・揚州で兵を集めていた**馬元義**を洛陽に送り込み、中常侍の**封諝・徐奉**らを内応させ、**3月5日**に内外呼応して蜂起する計画だった。

しかし張角の弟子・**唐周**が宦官に密告し計画が露見。馬元義は**車裂き**に処された。張角はやむを得ず予定を繰り上げて蜂起した。

#### 皇甫嵩・朱儁の鎮圧

| 将領 | 官職 | 戦績 |
|------|------|------|
| **皇甫嵩** | 左中郎将 | 長社の戦いで火計を用いて波才を撃破。広宗で張梁を斬る。張角の屍を引きずり出して晒す |
| **朱儁** | 右中郎将 | 皇甫嵩と共に潁川方面で黄巾軍を討伐 |

10月、皇甫嵩が広宗で黄巾軍を奇襲し張梁を斬首。**この時すでに張角は病死**しており、遺体を掘り出して首を晒した。

#### 劉備・関羽・張飛の義勇軍参加

正史『三国志』蜀書先主伝によれば、劉備は涿郡の**鄒靖**の下で義勇軍を結成し、黄巾の乱の討伐に参加した。関羽・張飛と共に功績を挙げ、**安喜県の尉**（県の軍事担当者）に任じられた。

**正史には「桃園の誓い」の記述はない**。ただし「恩若兄弟」（兄弟のような恩情をかけた）「寝則同床」（寝る時は同じ寝台で寝た）という記述があり、三人の親密さは正史でも認められている。

#### 曹操の初陣

正史『三国志』魏書武帝紀:

> **光和末、黄巾起。拝騎都尉、討潁川賊。**

（光和の末年、黄巾が起こる。騎都尉に任ぜられ、潁川の賊を討った。）

曹操は皇甫嵩・朱儁の指揮下に入り、潁川での討伐戦で功績を挙げ、**済南国の相**に任命された。

**ゲーム用ストーリー改善提案**: 現行の第1章introは董卓暗殺まで含んでいるが、史実では黄巾の乱（184年）と董卓暗殺（192年）は8年の隔たりがある。第1章は黄巾の乱にフォーカスし、董卓関連は第2章に回すのが史実に忠実。

### 第2章: 反董卓連合（189-192年）

#### 何進暗殺と宦官誅殺（189年）

**大将軍・何進**は外戚の筆頭として、宦官（十常侍）の排除を企てた。しかし妹の何太后が宦官の側に立ったため計画は難航。何進は涼州の**董卓**を洛陽に召喚して圧力をかけようとした（**袁紹の進言**）。

だが宦官たちは先手を打ち、**何進を宮中に呼び寄せて暗殺**した。これに激昂した袁紹・袁術らが宮中に突入して**宦官2000余人を皆殺し**にした。この混乱の中、少帝と陳留王が宮中から逃げ出し、そこに董卓が到着して両者を保護した。

#### 董卓の暴政

董卓は何進の軍を吸収し、さらに**丁原の部下・呂布を寝返らせて丁原を殺害**させ、一気に軍事力を掌握した。

董卓の主な暴挙:
- **少帝を廃位**し、陳留王を即位させた（献帝）
- 何太后を永安宮に幽閉し、のちに殺害
- **弘農王（廃された少帝）を毒殺**
- 反対派の官僚を次々と処刑
- **洛陽から長安への強制遷都**（190年）
- 遷都時に歴代皇帝の陵墓を盗掘し、宮殿・民家を焼き払った

#### 反董卓連合の結成と瓦解（190年）

袁紹を盟主として、各地の群雄が連合軍を結成。しかし実際には:
- **各勢力が保身に走り、積極的に戦おうとしなかった**
- 連合軍の内部で孫堅と袁術の対立など不和が多発
- 曹操は積極的に董卓軍と戦ったが寡兵で敗北

結局、反董卓連合は瓦解し、群雄はそれぞれの領地争いを始めた。

#### 呂布と董卓の確執、王允の計略（192年）

呂布が董卓を殺害した背景（正史）:
1. 董卓は些細なことで激怒し、**呂布に手戟（小型の戈）を投げつけた**ことがあった
2. 呂布は密かに**董卓の侍女と通じて**おり、発覚を恐れていた
3. 司徒の**王允**が呂布の不安を察し、董卓暗殺を持ちかけた

呂布は当初、「父子の関係がある」として拒否したが、王允は「お前は董卓の血縁ではない。手戟を投げつけられた恩義がどこにある」と説得した。

**192年4月**、王允は朝議の名目で董卓を宮中に呼び出し、**呂布が詔を奉じて董卓を刺殺**した。

**正史と演義の違い**: 『三国志演義』で有名な**「貂蝉」と「連環の計」は完全な創作**。正史には貂蝉という名の女性は登場せず、呂布が通じていたのは「董卓の侍女」（名前の記載なし）である。

---

### 第3章: 群雄割拠（193-200年）

#### 曹操の兗州確保と呂布との戦い

192年、曹操は兗州牧となり東郡を拠点とした。194年、曹操が徐州の陶謙を攻めている間に、**呂布が張邈・陳宮の裏切りにより兗州に侵入**。曹操は本拠地を失いかけるが、荀彧・程昱の奮闘で何とか三城を確保し、最終的に呂布を兗州から追い出した。

#### 孫策の江東平定（195-200年）

孫策は父・孫堅の死後、袁術の下に身を寄せていた。195年、袁術から兵を借りる形で江東への進軍を開始。

- 兵力は当初わずか**1000人余**だったが、進軍するにつれて**5000人以上**に膨張
- **周瑜**が歴陽で合流し、全面的に支援
- 劉繇、王朗、厳白虎、許貢らを次々と撃破
- **電撃的に江東一帯を平定**

陳寿の評:

> 孫策は傑出した英気を具え、その勇猛さと鋭敏さは並ぶ者がないほどであった。しかし**孫策は軽佻果躁であったがゆえに、身を滅ぼした**。

200年、**許貢の残党**に襲撃されて重傷を負い、弟の孫権を後継に指名して死去。享年26。

#### 劉備の流浪

劉備の流浪は正史に明確に記録されている:

| 時期 | 寄寓先 | 経緯 |
|------|--------|------|
| 184-191 | 各地転戦 | 黄巾の乱で義勇軍。安喜尉→高唐令→公孫瓚の下 |
| 191-194 | **公孫瓚** | 平原の相に任じられる |
| 194 | **陶謙** | 徐州救援に赴き、陶謙から丹楊兵4000を受ける。陶謙の死後、徐州を継承 |
| 196 | **呂布に奪われる** | 呂布が徐州を奪取。劉備は小沛に退く |
| 196-198 | **曹操** | 呂布に追い出され曹操に合流。曹操と共に呂布を討伐 |
| 199-200 | **袁紹** | 曹操に叛旗を翻して敗北、袁紹の下に逃亡 |
| 200-208 | **劉表** | 袁紹の下を離れ、荊州の劉表を頼る |

#### 袁術の僭称と没落（197-199年）

197年、袁術は**「仲」という国号で皇帝を僭称**した。「伝国璽を得た」「代漢者は当に塗高なるべし」（漢に取って代わる者は「塗」の位にある者だろう。「塗」は「途」に通じ、袁術の字「公路」と関連する）という讖緯を根拠とした。

しかし即位後:
- 孫策が離反
- 曹操に攻撃される
- 配下が次々と離散
- 最終的に**蜂蜜を求めて得られず、嘔血して死亡**（199年）

#### 呂布の処刑（下邳の戦い、198-199年）

198年冬、曹操は劉備と共に呂布を攻めた。

- 曹操は**泗水と沂水を引いて下邳を水攻め**
- 呂布の部将・**侯成**が陳宮・高順を捕らえて降伏
- 呂布は白門楼に登ったところで降伏

処刑の場面（正史）:
- 呂布は「縛り方がきつすぎる。少し緩めてくれ」と訴えた
- 曹操は「虎を縛るには堅くせねばならぬ」と答えた
- 呂布は曹操に「私が騎兵を率い、公が歩兵を率いれば天下は平定できる」と助命を願った
- **曹操が少し心を動かしたところ、劉備が「丁原と董卓の末路をお忘れですか」と諫めた**
- 呂布は「この大耳（劉備）こそ最も信用ならぬ者だ」と叫んだ
- **建安3年12月24日（199年2月7日）、呂布は縊死**させられた

---

### 第4章: 官渡の戦い（200-207年）

#### 官渡の戦いの詳細

200年、中原の覇権を巡って曹操と袁紹が激突した。

**兵力**: 正史本文では袁紹軍10万（騎兵1万）、曹操軍1万弱とされるが、裴松之はこの数字に疑問を呈している。

**戦いの経過**:

| 月 | 出来事 |
|----|--------|
| 2月 | 袁紹、白馬を攻撃。曹操は**関羽に命じて顔良を斬首**させ、白馬の包囲を解く |
| 8月 | 袁紹軍、官渡に集結。曹操は兵糧不足に悩む |
| 10月 | **許攸が袁紹陣営から曹操に寝返る** |
| 10月 | 許攸の情報により、曹操は**烏巣の兵糧集積地を奇襲** |
| 10月 | **淳于瓊**が守る烏巣を、曹操自ら歩騎5000で強襲・火をかけて壊滅 |
| 同日 | 袁紹軍の張郃・高覧が曹操に寝返る |
| 同日 | 袁紹軍壊滅。袁紹は800騎で逃走 |

**戦後**: 曹操は袁紹陣営から押収した味方の密通書簡を**読まずに全て焼却**した。「袁紹の勢い盛んなるとき、私さえ自分を守り切れなかったのに、他の者を責められようか」と述べた。一方、袁紹は獄中から正しい助言をしていた田豊を斬った。

#### 袁紹の死と袁家の滅亡

202年、袁紹は病死。後継を巡って長男・**袁譚**と三男・**袁尚**が争い、曹操はこの内紛につけ込んで各個撃破した。207年までに袁家は完全に滅亡。

#### 三顧の礼と隆中対（207年）

劉備は荊州の劉表を頼って新野に駐屯していた。徐庶の推薦で諸葛亮の存在を知り、**三度その草廬を訪問**した。

隆中対（隆中策）は正史『三国志』蜀書諸葛亮伝にほぼ同文が記録されている:

諸葛亮の戦略骨子:
1. **荊州と益州を領有**して根拠地とする
2. 西は諸戎と和し、南は越（南蛮）を撫で、**外は孫権と結ぶ**
3. 天下に変化あれば、**一軍を荊州から宛・洛に向かわせ、一軍を益州から秦川に進ませる**
4. これにより**天下三分**を実現し、最終的に漢室を復興する

#### 曹操の北方統一

207年、曹操は北方の**烏桓**を征伐し、**白狼山の戦い**で烏桓単于の蹋頓を張遼が斬り、北方を完全に平定。ここに中原統一が完成した。

---

### 第5章: 赤壁の戦い（208-209年）

#### 曹操の南征

208年7月、曹操は荊州を攻めるため南征を開始。8月に劉表が病死し、跡を継いだ**劉琮が9月に降伏**。劉備は荊州を追われ南方へ逃亡。

#### 孫劉同盟の成立

正史には同盟成立の経緯について二つの記述がある:

- **蜀書諸葛亮伝**: 諸葛亮が孫権との同盟を劉備に献策し、自ら使者として赴いた
- **呉書魯粛伝**: 魯粛が劉表の弔問を名目に南下し、劉備と接触。劉備が諸葛亮を使者として孫権のもとに派遣

孫権の朝廷では降伏論が大勢を占めていたが、**魯粛の進言**と**諸葛亮の説得**、そして**周瑜の断固たる抗戦論**により、孫権は開戦を決意した。

周瑜の抗戦論（正史）: 周瑜は曹操軍の弱点を的確に分析し、「北方の兵は水戦に慣れず、荊州の兵は降伏したばかりで心服していない。疫病も流行している。必ず勝てる」と主張した。

#### 赤壁の戦い（正史での実態）

正史での赤壁の戦いは、『三国志演義』ほど劇的には描かれていない。

**魏書武帝紀**の記述は極めて簡潔:

> 公至赤壁、与備戦不利。於是大疫、吏士多死者、乃引軍還。

（曹操は赤壁に至り、劉備と戦うが不利であった。ここに大いに疫病が流行し、官吏兵士で死ぬ者が多かったので、軍を引き返した。）

**呉書周瑜伝**はより詳しい:
- 黄蓋が苦肉の計を献策:「敵船は互いに繋がれている。火を以て攻めるべし」
- 黄蓋は火のついた船を曹操軍の船団に突入させた
- 延焼は岸辺の陣営にまで及んだ
- 曹操軍は大敗して撤退

**最大の敗因は「疫病」**であり、火攻めはそこに追い打ちをかけたものとされる。

**正史と演義の違い**:
- **諸葛亮の「東風を呼ぶ」は完全な創作**（正史にない）
- **蒋幹の盗書も創作**
- 正史での赤壁の主役は**周瑜**であり、諸葛亮は孫権の説得が主な功績
- 「十万本の矢」（草船借箭）も創作

#### 荊州分割

赤壁後、荊州は三分された:
- 曹操: 荊州北部（南陽郡など）
- 劉備: 荊州南部四郡（武陵・長沙・桂陽・零陵）
- 孫権: 南郡（周瑜が攻略）

のちに劉備が孫権から南郡を「借りる」形で荊州の大部分を支配。この「荊州の貸借」問題が後の孫劉対立の火種となる。

---

### 第6章: 三国鼎立（210-222年）

#### 劉備の益州入り（211-214年）

211年、益州の劉璋が張魯の脅威に対抗するため劉備を招き入れた。法正と張松が内応し、劉備は徐々に軍を進めた。

212年、劉備と劉璋の関係が決裂。劉備は益州攻略を開始。**龐統が雒城攻めで戦死**した（流れ矢に当たる）。諸葛亮が張飛・趙雲と共に増援に赴き、214年に成都を包囲。劉璋は降伏し、劉備は益州を領有した。

#### 漢中の戦い（217-219年）

215年に曹操が漢中の張魯を降伏させ、漢中を領有。法正が漢中の重要性を説き、劉備に攻略を進言。

**定軍山の戦い（219年）**:
- 劉備は「声東撃西」の戦術を用い、まず張郃の東方面を攻撃
- 夏侯淵の西方面の柵を火攻めにした
- 消火のために出陣した夏侯淵を、法正の指示を受けた**黄忠が突撃して斬殺**
- 益州刺史の趙顒も討ち取られた

この勝利により劉備は漢中を制し、**漢中王**を称した。

#### 関羽の樊城攻め（219年）

劉備の漢中王即位に呼応し、関羽は荊州から北上して**曹仁の守る樊城を攻囲**した。

**于禁の敗北と降伏**: 曹操は救援として于禁に七軍を率いさせたが、折からの長雨で漢水が氾濫。**七軍は水没し、于禁は降伏**。この出来事は「**威震華夏**」（天下が震撼した）と記される。曹操は一時、許都からの遷都すら検討した。

#### 呂蒙の白衣渡江

呂蒙は病気を理由に引退を装い、若き**陸遜**を後任に推薦。関羽は陸遜を軽んじて荊州の守備兵力を前線に移動させた。

**白衣渡江**: 呂蒙は兵士たちに**白い服を着せて商人に偽装**させ、船で荊州に向かった。「白衣」は当時の商人の衣装。**虞翻**の活躍もあり、関羽配下の**士仁・糜芳**があっけなく降伏。公安と南郡が陥落した。

#### 関羽の死

呂蒙は関羽の部下の家族を厚遇し、わざと使者に見せた。これを知った関羽の部下たちは敵対心を失い、**関羽の軍は内部から瓦解**した。

**建安24年12月**、関羽は当陽県の臨沮において退路を断たれ、関平と共に捕らえられて処刑された。

#### 夷陵の戦い（221-222年）

正史蜀書先主伝には明確に記されている:

> 東を征して関羽の仇を報いんとした

劉備は蜀漢皇帝として自ら出陣。趙雲は反対し、諸葛亮も阻止できなかった。

**戦いの経過**:
- 劉備は巫から夷陵にかけて**50以上の陣地**を築いた
- 陸遜は半年以上守りに徹し、将兵の不満を抑えつけた
- **222年6月**、陸遜は蜀軍の油断を見て総攻撃。**火計**を用いて蜀軍の連営を焼き払った
- 蜀軍は壊滅的打撃を受けた

#### 劉備の死と白帝城の託孤（223年）

劉備は白帝城に退き、病に倒れた。孫権が和議を申し入れ、劉備はこれに応じた。

**223年4月**、劉備は諸葛亮を呼び寄せ、遺詔を残した:

> **君才十倍曹丕、必能安国、終定大事。若嗣子可輔、輔之。如其不才、君可自取。**

（君の才は曹丕の十倍、必ず国を安んじ、大事を定められるだろう。もし後嗣（劉禅）が輔佐するに値するなら輔佐せよ。もし才がないならば、**君が自ら取れ**。）

諸葛亮は涙を流して「臣は股肱の力を竭くし、忠貞の節を効し、死して後已むのみ」と誓った（のちに出師表の「鞠躬尽瘁、死而後已」に結実する）。

---

### 第7章: 北伐（223-234年）

#### 諸葛亮の南征（225年）

正史『三国志』蜀書諸葛亮伝では、南征は比較的簡潔に記される。朱褒・雍闓・高定らの反乱を鎮圧し、南中を平定した。

**「七擒孟獲」について**: 正史『三国志』本文には孟獲の名前すら登場しない。「七度捕らえて七度放した」という逸話は**『漢晋春秋』**や**『華陽国志』**に記載があるが、『三国志演義』で大幅に脚色された。演義に登場する祝融夫人、木鹿大王などは**完全な創作**。

歴史上の孟獲は南中の豪族であり、「蛮夷の王」ではない。**孟氏は南中四大姓の一つ**で、漢人系の有力氏族であった。

#### 出師表（227年）

建興5年（227年）、諸葛亮が第一次北伐に際して劉禅に奉った上奏文。中国文学史上の名文とされる。

冒頭の有名な一節:

> **臣本布衣、躬耕於南陽、苟全性命於乱世、不求聞達於諸侯。先帝不以臣卑鄙、猥自枉屈、三顧臣於草廬之中、諮臣以当世之事、由是感激、遂許先帝以駆馳。**

（臣はもと布衣にして、南陽にて躬耕し、苟も乱世に性命を全うし、諸侯に聞達を求めず。先帝は臣の卑しきを以てせず、みだりに自ら枉屈し、三たび臣を草廬の中に顧み、臣に当世の事を諮りたまい、これに由りて感激し、遂に先帝に許すに駆馳を以てす。）

結びの誓い:

> **鞠躬尽瘁、死而後已。**

（身を粉にして力を尽くし、死して後やむ。）

#### 第一次北伐と街亭の敗戦（228年）

228年春、諸葛亮は第一次北伐を開始。天水で**姜維**が降伏して蜀に帰順（これは正史にも記録あり）。初期は順調に進んだが、**街亭の敗戦**で全てが瓦解した（詳細は第1章の司馬懿の項を参照）。

#### 五丈原の対陣と諸葛亮の死（234年）

（司馬懿の項 1-5 で詳述済み）

#### 司馬懿の台頭

諸葛亮の死後、蜀漢の脅威は大幅に低下した。一方で司馬懿は:
- 遼東の公孫淵を討伐（238年）
- 曹叡の崩御時に顧命大臣（239年）
- 高平陵の変で魏の実権を掌握（249年）

と着実に権力を拡大していった。

### 第8章: 天下統一（235-280年）

#### 司馬懿の高平陵の変（249年）

（司馬懿の項 1-7 で詳述済み）

#### 司馬師・司馬昭の権力掌握

**司馬師**（字は子元）:
- 司馬懿の死後（251年）、撫軍大将軍として全権を掌握
- 254年、皇帝曹芳を廃立し、曹髦を擁立
- **毌丘倹・文欽の反乱**を鎮圧するが、この戦いで負傷し255年に死去

**司馬昭**（字は子上）:
- 兄・司馬師の後を継いで大将軍に就任
- **260年、皇帝曹髦が司馬昭排除のクーデターを試みるが失敗し、殺害される**（「司馬昭之心、路人皆知也」＝司馬昭の野心は道行く人さえ知っている、という曹髦の有名な言葉）
- 蜀漢を滅ぼした功績で**晋王・相国**に封じられる
- 265年に病死

陳寿は晋の臣下として正史を記述したため、司馬氏の悪行は**意図的に曖昧に書かれ、「行間を読ませる」手法**が採られている。

#### 姜維の北伐

諸葛亮の遺志を継いだ姜維は、238年から262年にかけて**6回から8回**の北伐を実施した（「11回」は一部の数え方による）。

主な戦績:
- **延熙16年（253年）**: 費禕が暗殺された後、軍権を完全に掌握
- **延熙19年（256年）**: **段谷の戦い**で鄧艾に大敗。蜀の国力を大いに疲弊させた
- **景耀5年（262年）**: 最後の北伐。鄧艾に撃退される

姜維は軍事的才能はあったものの、連年の出兵で蜀漢の国力を消耗させた面も否めない。宦官の**黄皓**の専横もあり、蜀漢の内政は腐敗していた。

#### 鄧艾の陰平越え（263年）

263年夏5月、魏は蜀漢討伐を開始。

| 将領 | 兵力 | 進路 |
|------|------|------|
| **鄧艾** | 3万余 | 陰平から迂回 |
| **鍾会** | 10万余 | 漢中を正面攻撃 |
| 諸葛緒 | 3万余 | 橋頭を封鎖 |

**鄧艾の陰平越え**: 鍾会と姜維が剣閣で対峙している間、鄧艾は**陰平の険路**を越えて成都盆地に侵入した。摩天嶺を越え、兵士たちを毛布に包んで崖を滑り降ろすという壮絶な行軍であった。

- **江油を陥落**
- **涪県を攻略**
- **綿竹の戦い**で諸葛瞻（諸葛亮の子）を討ち取る
- 成都に迫る

#### 蜀漢滅亡（263年）

鄧艾が成都に迫ると、劉禅は譙周の進言により**降伏を決意**。棺を載せた車に自らを縛り付けて降伏した。

姜維は劉禅の降伏命令を受け、**鍾会に降伏**。しかし内心では復興を企て、鍾会の野心を利用して反乱を画策した。最終的に反乱は失敗し、**姜維・鍾会ともに殺された**。鄧艾も鍾会の讒言で逮捕され、護送中に殺された。

#### 司馬炎の晋建国（265年）

265年12月、司馬炎は**曹奐に禅譲を迫り**、皇帝の位を奪って国号を「晋」、元号を「泰始」とした。賈充・裴秀・王沈・羊祜・荀勗・石苞・陳騫らが推戴した。

これにより、曹操が基盤を築き、曹丕が建国した魏は45年で滅亡。**三国のうち二つが消え、残るは呉のみ**となった。

#### 羊祜・杜預による呉征伐（279-280年）

**羊祜**（字は叔子）:
- 荊州都督として呉との国境を守り、**徳を以て呉の民を懐柔する戦略**を採った
- 対岸の呉将・陸抗との間に「紳士的な戦争」とも言うべき関係があった
- 278年、呉攻略の遺志を残して病没。呉攻略の計画を詳細に策定

**杜預**（字は元凱）:
- 羊祜の後任として呉征伐の総指揮を執る
- 279年、司馬炎は**6方向から呉を攻める**大規模侵攻を命じた
- **王濬**が蜀から長江を下る水軍を率い、杜預は襄陽から江陵を攻略
- 「破竹の勢い」という言葉は**杜預の言葉**に由来する

#### 呉滅亡（280年）

**太康元年（280年）3月15日**、晋軍20万が建業に迫り、最後の呉の皇帝・**孫皓が降伏**。

三国時代は96年間（184年の黄巾の乱から280年の呉滅亡まで）にわたる長い戦乱の後、ここに幕を閉じた。

しかし晋の統一は長続きせず、司馬炎の死後、**八王の乱**（291-306年）が勃発。さらに**五胡十六国**の大混乱へと続き、中国は再び分裂の時代に突入する。

## 3. 絆（Bond）の歴史的根拠

### 絆0: 桃園の誓い

**正史の記述**:

正史『三国志』蜀書関羽伝:

> **先主与二人寝則同床、恩若兄弟。而稠人広坐、侍立終日。**

（劉備は二人（関羽・張飛）と寝る時は同じ寝台で寝て、兄弟のような恩情をかけた。しかし大勢の前では、二人は一日中立って傍に侍した。）

つまり:
- **「桃園の誓い」という儀式は正史にない**。『三国志演義』の創作。
- しかし**三人の親密さは正史でも確認できる**（同床・恩若兄弟）。
- 公の場では主従の礼を守っていたという記述もあり、「義兄弟」というより「親密な主従」に近い。
- 関羽の死を聞いて劉備が呉を攻めたこと、張飛が暗殺されたことは史実。

**ゲーム用の絆テキスト改善案**: 「桃園で義兄弟の契りを結んだ」を「寝台を共にするほど兄弟のごとく親密であった」に正史準拠で修正できる。ただし、ゲームの華として演義のロマンを残すのも選択肢。

---

### 絆の歴史的根拠: 曹操と荀彧の関係

**現在のゲーム**: bonds.jsのid:11「五大謀臣」に荀彧（heroes配列のid:29）が含まれているが、曹操と荀彧の複雑な関係を独立した絆にする価値がある。

**正史の記述**:

荀彧は若くして「**王佐の才**」と称された潁川の名士。曹操の覇業を初期から支えた最大の功臣の一人。

しかし**建安17年（212年）**、董昭らが曹操の**魏公就任と九錫授与**を推進した際、荀彧はただ一人反対した:

> **「公が義兵を起こしたのは、本来朝廷を救い、国家を安定させるためである。真心からの忠誠を保持し、偽りのない謙譲さを守り通してきたのだ。君子は人を愛する場合徳義によるものだ。そのようなことをするのは宜しくない。」**

この反対により曹操との関係は決裂。

**荀彧の死の諸説**:

| 史料 | 記述 |
|------|------|
| **正史『三国志』本伝** | 「以憂薨」（憂いをもって没した）。享年50歳 |
| **『魏氏春秋』** | 曹操から贈られた**器（箱）の中身が空**であったのを見て、服毒自殺した |
| **『献帝春秋』** | 伏完からの反曹操の手紙を長期間黙秘していたことが発覚し、疎まれた |

「空箱」の解釈: 「お前にはもう食わせるものはない（＝もう用済みだ）」という暗示とされる。ただし信憑性は低い。

**本質的な対立**: 荀彧は**漢王朝への忠臣**として曹操を支えたのであり、曹操が漢を簒奪することは断じて認められなかった。一方の曹操にとって、荀彧の反対は覇業の最大の障害であった。20年以上の信頼関係が、皇帝の地位を巡って崩壊するという悲劇。

  heroes: [曹操, 荀彧],
  desc: '曹操と荀彧。20年の信頼を砕いた皇位への野心',
  lore: {
    short: '空の器に込められた意味を、荀彧は悟った',

  }

### 絆の歴史的根拠: 周瑜と諸葛亮

**現在のゲーム**: id:10「赤壁の炎」で諸葛亮・周瑜・黄蓋が同じ絆に含まれているが、周瑜と諸葛亮の関係は以下の通り。

**正史の記述**:

正史では周瑜と諸葛亮の間に**直接的な対立関係の描写はほとんどない**。

- **赤壁の戦いの主役は周瑜**。諸葛亮の役割は孫権を説得して開戦を決意させたこと
- 「既生瑜、何生亮」（周瑜の臨終の言葉「なぜ天は周瑜を生んだのに、諸葛亮も生んだのか」）は**完全に演義の創作**
- 正史の周瑜は**度量の広い人物**として描かれ、諸葛亮に嫉妬する器の小さい人物ではない

正史における周瑜の評価:

> 陳寿曰: **「周瑜は器量広大にして、雅量高致なり」**

**周瑜の実際の計画**: 正史では周瑜は諸葛亮ではなく、むしろ劉備の勢力拡大を警戒していた。周瑜は孫権に劉備を軟禁することを進言したが、孫権は採用しなかった。

**ゲーム用の修正提案**: id:10「赤壁の炎」の lore を、演義の「東風」ではなく「火計は黄蓋の献策、総指揮は周瑜」という正史準拠に修正できる。

### 絆の歴史的根拠: 関羽と張遼の友情

**正史の記述（非常に重要）**:

この逸話は**正史『三国志』に明確に記述がある**貴重な絆。

**経緯**: 198年、曹操が呂布を討伐した際、関羽は曹操に降伏・帰順した。曹操は関羽の人物に感服し厚遇したが、関羽の心が劉備から離れないことも察していた。

正史蜀書関羽伝:

> **曹公壮羽為人、而察其心神無久留之意、謂張遼曰：「卿試以情問之。」**

（曹操は関羽の人物を壮んとしたが、その心に久しく留まる意のないことを察し、張遼に「お前、気持ちを込めて聞いてみてくれ」と言った。）

張遼が関羽に尋ねると:

> **羽歎曰：「吾極知曹公待我厚、然吾受劉将軍厚恩、誓以共死、不可背之。吾終不留、吾要当立効以報曹公乃去。」**

（関羽は嘆息して言った。「私は曹公が私を厚遇してくださることをよく分かっている。しかし私は劉将軍の厚い恩を受け、共に死ぬと誓ったのだ。これに背くことはできない。私は最後まで留まらない。しかし必ず功を立てて曹公への恩に報いてから去るつもりだ。」）

張遼はこれを曹操に報告した。曹操は関羽を「義人」だと評した。

**白馬の戦い（200年）**: 関羽は曹操の先鋒として袁紹の大将・**顔良を万人の中で斬首**し、曹操への恩義に報いた。その後、賜物を全て封印して書状を残し、劉備のもとに去った。曹操の側近は追撃を進言したが、曹操は「各々主に仕えるもの」と追撃を許さなかった。

**ゲーム新規絆案**:
```
{
  name: '敵陣の友誼',
  category: 'sworn',
  heroes: [関羽, 張遼],
  desc: '関羽と張遼。敵味方を超えた武人の信義',
  lore: {
    short: '義を知る者同士、敵味方の別は問わぬ',
    full: '曹操は関羽の心を知りたいと張遼に託した。関羽は「劉備への恩義に背けない」と告白し、張遼はそれを曹操に正直に伝えた。二人は敵味方に分かれても、互いの義を認め合った。正史に記された、三国志で最も美しい友情の一つ。'
  }
}
```

### 絆6: 諸葛亮と司馬懿の対決

**正史の記述**:

二人の対決は主に**第四次北伐（231年）と第五次北伐（234年）**で記録されている。

**第四次北伐（231年）**: 上邽の戦い。局地的には蜀軍が勝利したが、食糧不足で撤退。張郃が追撃中に矢に当たり戦死（司馬懿が張郃に追撃を命じたが、張郃は反対していた）。

**第五次北伐（234年）**: 五丈原の対陣。100日余りの膠着。
- 諸葛亮は司馬懿に女性の髪飾り（巾幗婦人之飾）を贈って挑発
- 司馬懿は耐え忍んで出陣せず
- 諸葛亮の使者に対し、司馬懿は「孔明は睡眠が少なく、食事も少量だと聞く。長くはあるまい」と評した
- 234年8月、諸葛亮が五丈原にて病没

**二人の関係の本質**: 「宿命のライバル」という演義のイメージとは異なり、正史では**直接対峙したのはわずか2回**。しかし、攻める諸葛亮と守る司馬懿という対照的な戦術家の対決は、三国時代後期の最大の見せ場である。

**司馬懿の諸葛亮評価**: 蜀軍が撤退した後、司馬懿はその陣地跡を視察し、**「天下の奇才なり」**と感嘆したと伝わる。

**ゲーム用の絆テキスト補足**:
- 「空城の計」は**演義の創作**であり、正史にはない
- 五丈原の対陣は正史の記録
- 「死せる孔明、生ける仲達を走らす」は正史（裴松之注に引く『漢晋春秋』）にも記録あり

## 付録: ストーリーテキスト改善のための正史準拠チェックリスト

現行のstory-text.jsに対する正史準拠の指摘:

### 第1章
- **問題**: 「桃園にて義兄弟の契りを結び」→正史にはない。
- **提案**: 「関羽・張飛と兄弟のごとき固い絆を結び」に修正可能。ただしゲームの演出として残すのも選択肢。

### 第3章（赤壁）
- **問題**: 「百万の大軍」→正史では曹操軍は20万程度。「百万」は演義の誇張。
- **問題**: 「大喬・小喬を奪い、銅雀台に囲うでしょう」→**これは完全に演義の創作**。正史の周瑜の抗戦論は軍事的分析に基づいている。
- **提案**: 正史準拠にするなら周瑜の軍事分析を引用。演義の名場面として残すのも選択肢。

### 第6章（出師表）
- **問題**: 「空城の計で司馬懿を退ける」→**正史にはない。演義の創作**。
- **問題**: 街亭で司馬懿を相手にしたのは**張郃**であり司馬懿ではない。
- **提案**: 正史準拠にするなら「街亭では馬謖が張郃に敗れ」に修正。

### 第7章（姜維の北伐）
- **問題**: 「十一度の北伐」→回数は数え方により異なる。正確には6～8回程度。
- **提案**: 「幾度にもわたる北伐」のような曖昧な表現にするか、「数次の北伐」とする。
